SETTING UP YOUR WEBSITE

Click on Settings

1) ICO/STO settings




ICO/STO Settings

Token name: Enter name of token.

Token symbol: enter token symbol like BTC, ADA, ETH.

Price/USD: enter price amount for per token.

---Purchase & Additional Setting: check the prefer currency/means for purchase of token.

2) Website Settings: what are to be display on front-end





Site Name: input the name of your site e.g lulo.

Site Email: input email for which user will be able to reach out to admin e.g lulo@support.com.

Contact Address: permanent address of token.

URL address: input your url/website address.

Site phone number: input website phone number.

Click save settings button.

3) Referral Settings





Add referral commission in percentage as shown above,

Then save.

How to add new payment method


click on Add info button.
