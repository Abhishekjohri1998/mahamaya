Getting Started

Introduction to PRECHAIN (ICO management system)

Prechain (ICO management system) helps companies/individuals to manage and organize their fund raising (ICO) process, making it seemless and transparent.

ICO is a type of capital-raising activity in the cryptocurrency and blockchain environment, ICO can be viewed as an initial public offering (IPO) that uses cryptocurrencies.


Prechain (ICO management system) will provide you with admin dashboard to setup your token (price and ICO stages) and seemless management at all levels of your token offering.

AT A GLANCE

Admin setup the system.

Admin create token.

Users create account and login to dashboard.

Users buy token with quantity using the available swap methods.

Users holds token still lunch date.

Users can buy token at different level of progress.

Users can even stake token from the ICO dashboard to earn more.
